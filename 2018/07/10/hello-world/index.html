print()函数

2、print()函数
语法格式：print(value,…,sep=’ ‘,end=’\n’)
其中，value代表我们要输出的内容，可以是字符串，可以是数值等内容

3、print()函数的基本使用
print()常用输出方式：
print(‘hello ‘)、print(123456)等
参数格式输出
%s：代表了字符串内容，其中
print(‘welcome %s’%Beijing)=>welcome Beijing
print(‘welcome %10.3s’%Beijing)=>welcome Bei,输出结果为10个字符其中包含字符串Beijing中前三个，其余用空格填充
%d：代表了整数内容，其中
print(‘price %d’%10)=>price 10
print(‘price %d’%10.5) => price 10
print(‘price %3d’%10000)=> price 10000
注意这里面的3d表示输出一个宽度为3个单元的数值，如果数值长度小于三，那么数值靠右对齐。
-3d表示，数值反向对齐（靠左对齐）
%f：代表了浮点数内容(注意，在Python中浮点数是一个近似值，所以结果也是近似)
print(‘price %f’%10.2)=>10.200000
print(‘price %.1f’%10.2) => 10.2
print(‘prince %.1f’%10.58) = >10.6 当取一位小数位的时候，将小数位第二位进行舍入操作，但是Python的中间值不是5，导致当小数位第二位是5时，直接被舍去
print(‘%5.1f’%10.3) => 10.3
输出一个宽度为5的单位，其中整数3位，小数部分1位，小数点占一位

变量在内存中的存在情况
1）python中的变量，在声明和赋值时，不仅声明了变量的名字同时还为这个变量指向了内存中的地址，该地址用来存储变量的值。
2）python中整数、字符串、浮点数等其本身都属于常量，所以当赋值给一个变量时，如：
i = 10意味着将10这个常量在内存中的地址指向，因此，i = 10 ，j = 10 使用id(i),id(j）我们法向他们的id是相同的。
因为，他们的值10在内存中的地址时相同的。

3）变量与内存地址
当我们声明一个变量时，对其进行赋值。实际上就是将变量值所在的内存地址，赋值给该变量。也就是说这个变量指向其值所在的内存地址。
当我们改变变量的值的时候，说白了就是将变量指向的内存地址进行变化。
举例：我们日常购买商品，快递员通过你的邮寄地址将快递送到你本人手里。可以理解为，快递员拥有指向你这个变量的地址，通过地址，将快递商品赋值给你。
4）变量与常量
变量：我们认为其内容是临时结果，可变的。供后续的程序代码进行调用。
常量：我们认为其内容是不变的，或者在程序运行的过程中也不发生改变。
在Python中，我们通过大写变量名，来表明这是一个常量，比如：CARDID = ‘12048377293’
5)变量命名规则
首先，Python中的变量命名，必须由字母、数字或下划线三者组成，不能有其他的字符出现
其次，变量名的首个字符，必须是字母或下划线
然后，Python中的关键字不能作为变量名来使用，常用的关键字诸如：if,for,is,none等等。
Python是严格区分大小写的语言，其中A = 10 ，a = 10。我们认为这是两个不同的变量

字符串常用的方法

$ 4、字符串常用的方法
1）find() 查找
函数格式：str.find(子字符串[,开始索引位，结束索引位])
返回值：如果查找成功，则返回所查找内容在字符串中的索引位置，如果没有找到，返回-1.
注意：如果所查找的内容中，有多个相同的字母，只会返回第一个字母所在的索引位。比如：
str1 = ‘hello’
str1.find(‘ll’) => 2 该例中，’ll’在字符串中出现了两次，但只会返回第一次出现的索引位置。
为什么查找时，没有找到目标字符串的情况下会返回-1。其实，在find操作中，会对字符串进行扫描从字符串的索引位0开始，一直到字符串结尾，如果在字符串最后一位没有找到匹配的内容，将指向字符串结束符(EOF)，该结束符为-1。所以找不到时，find返回值为-1.
更多用法：
string1 = ‘apple,banana,apple,apple’

string1.find(‘apple’)
0
string1.find(‘apple’,5)
13
string1.find(‘apple’,7,20)
13
string1.find(‘apple’,14,20)
-1
string1.find(‘apple’,14,24)
19

对比：
index() 返回指定字符的索引位
函数格式：index(子字符串[,开始索引位，结束索引位])
返回值：如果找到指定的字符串，则返回其索引位；如果找不到就报错ValueError

find()与index()函数的区别
第一，两者在执行结果时类似的，如果找到指定的字符就会返回其索引位
第二，find()如果找不到指定的字符时，返回-1；而index()找不到指定的字符，raise ValueError.
第三，从效率上来讲，index的查询速度要优于find,因为在python中对于像字符串这样的有序数据类型，会产生一个索引对象来自动维护其序列，find()函数不管是否有这样的序列，都需要在字符串本身进行逐一查找，而index()会通过这个索引对象的映射方式，来快速找到对应的索引位。
2）split() 拆分字符串
函数格式：str.split([分隔符])
返回值：根据指定的分隔符，对字符串进行拆分，返回一个列表；如果不指定分隔符，那么会将整个字符串，作为列表的一个元素。例如：
string1 = ‘tv,pc,notebook,phone’

string1.split(‘,’)
[‘tv’, ‘pc’, ‘notebook’, ‘phone’]
string1.split()
[‘tv,pc,notebook,phone’]
注意：split中的分隔符，我们根据字符串中的实际情况而定。如果使用了不正确的分隔符，str.split(‘错误分隔符’)操作结果str.split(),会将整个字符串作为列表的一个元素。
3）strip() 字符串左右两端去除指定字符（默认情况，去除空格）
函数格式：str.strip([指定字符])
返回值：根据指定的字符，对字符串进行两端去除其内容，返回一个新的字符串。如果字符串中间有空格或指定字符，那么该函数不起作用。如果指定的字符在字符串左右两端不存在，那么将返回字符串本身。
例如：
缺省情况
string1 = ‘ hello world ‘
string1.strip()
‘hello world’
print(string1)
hello world

指定字符
string1 = ‘awelcom to zhengzhoua’

string1.strip(‘a’)
‘welcom to zhengzhou’
注意：
1）字符串类型，一旦被声明赋值后，其内容是不可被改变的，如果需要改变原字符串变量的值，必须对其进行重新赋值。
2）如果指定字符内容为一个字符序列，那么意味着使用strip去除左右两端指定字符时，只要符合字符序列中的某一位，就进行去除
string1 = ‘ awelcom to zhengzhouab’
string1.strip(‘ ab’)
‘welcom to zhengzhou’

字典数据类型，由key=value结构构成的。也就是说字典类型中的每个元素都是有两部分构成，中间使用：连接。其中，key和value的可以是任意数据类型的内容。

3、字典的特性

然后，字典中的value需用通过key来获取。

3、迭代器（Iterator）
1）概念：迭代器访问元素，是从第一个元素开始，依次访问后续元素直到结束。其中迭代器只能向前不能回退。其包含两个方法iter()和next()。
iter()用来创建一个迭代器，next()用来访问下一个元素。
目前我们所学的字符串、列表、元组都可以创建迭代器，那么他们也是可迭代化的对象。
迭代器两个步骤：
1、当调用next()函数，会将迭代器当前位置状态进行改变
2、会将下一个元素进行返回

迭代器的延迟作用：迭代器生成的元素并不是在迭代器声明时，就准备好了的。通常时在调用迭代器需要提供该元素时生成的，也就是说只有在我们需要的时候迭代器才产生相关元素，不需要时并不会主动的生成相关元素。我们可以把迭代器看成一个容器，到我们需要时使用next()方法返回容器中的下一个值。
迭代器实现了工厂模式的对象，它在你每次访问下一个值的时候给你返回。

可迭代化（iterable）：表明相关的对象可以创建迭代器对象，同时他们自身都是可以进行for数据遍历的。我们可以使用isinstance()来判断对象是否为可迭代化

总结：迭代器和生成器
1、容器是一系列的元素集合，str、list、set、dict对象都可以看作是容器，容器都可以进行迭代（即用在while或for循环），凡是这样的容器我们都称为可迭代化对象（Iterable）。
2、可迭代化对象都实现了iter方法，该方法返回一个迭代器对象。
3、迭代器都拥有一个状态字段，用于记录下次迭代返回值。它实现了next和iter方法，迭代器不会一次性把所有的元素都加载到内存中，而是在需要的时候才生成返回结果。
4、生成器是一种特殊的迭代器，它的返回值不是通过return来返回的，而是通过yield来返回的。

copy()(浅拷贝)和deepcopy()（深拷贝）
1、浅拷贝：我们使用copy()函数将列表中的元素，进行复制，通常对于普通的列表（一维）能够实现完全的值传递，即将自身的副本赋值给另外一个列表。例如：
ls1 = [1,2,3,4,5]
ls2 = ls1.copy() # 将ls1中的元素赋值给ls2,此时他们的值相等的
ls1[3] = 6 #此时，对ls1中索引位3的值进行修改，变为6
观察其结果，我们可以发现ls1 => [1,2,3,6,5],但是ls2 =>[1,2,3,4,5]没有发生改变
一旦，ls1的列表元素结构发生变化，出现子列表嵌套。
如：ls1 = [1,2,3,[‘one’,’two’],5]，这是再使用copy()对ls2赋值，我们进行如下修改：
ls1 = [1,2,3,[‘one’,’two’],5]
ls2 = ls1.copy()
ls1[3][0] = ‘three’
我们观察其结果，可以发现ls1 => [1,2,3,[‘three’,’two’],5],同时ls2也会随之变化 ls2 => [1,2,3,[‘three’,’two’],5]。
总结：这种现象我们成为浅拷贝，copy()函数进行组合数据类型拷贝时，只能控制第一层元素进行值传递，一旦出现嵌套的子列表、子字典等内容，将不再拷贝其内容的值，而是直接将这个嵌套的内存地址，赋值给ls2。所以会造成一旦原列表改变子列表内容后，目标列表中对应的子列表也发生改变。
2、深拷贝：我们需要引入copy这个模块，调用deepcopy()函数。写法如下：
import copy

ls1 = [1,2,3,[‘one’,’two’],5]
ls2 = copy.deepcopy(ls1)
ls1[3][0] = ‘three’
此时，我们发现ls1内容发生改变后，但是ls2并没有随之改变，那么也就是意味着deepcopy()将ls1中内容，包括嵌套里面的元素都进行了副本赋值，给ls2。因此，他们只是值相同，并没有指向同一个内存地址。

2、sorted()函数提供排序功能，但是该排序进行了临时性排序，即只是产生一个有序结果，并不会真正改变原列表中元素的位置。

9、函数
一、自定义函数
1、函数的定义
函数的格式：
def 函数名([参数列表]):
执行代码
其中，函数名的命名方式与变量名的要求相同，即只能是数字、字符、下划线三者构成，同时不能由数字开头。
另外，函数的参数根据情况，可以有参，也可以无参。同时，参数的个数，可以是1到N个。最后，调用函数时，一定要保证该函数已经被定义。调用的函数名与定义的函数名一致。
2、函数的调用
函数在外部调用时，直接写函数名以及函数参数即可，不需要将函数体写出来。同时，函数是否有返回值，由其函数体最后一句代码是否为return来决定的。如果由return语句，那么就意味着该函数有返回值，如果没有返回值就意味着返回值为None。
注意：函数使用return作为返回值语句，该命令通常放在函数的最后一句，保证函数相关的代码都执行完毕后才返回，如果后面还有其他程序，则不会被执行。

3、函数的参数
概念：函数在定义时声明的参数，我们称为形参（形式参数）。形参的作用，主要是用来表明参数顺序、类型和个数。并不代表具体内容。
我们在调用函数的时候，根据形参的规定（顺序、类型、个数），传递进来实际的内容。我们称为实参。
例如：
def display(name,age): # 定义函数时声明的形参：name,age
print(name,age)

display(‘tom’,22) #调用函数时实际传递的参数值：tom, 22
1）必须的参数
我们在函数定义时，显示声明在函数的参数列表中的参数，这些参数一旦被声明，函数在调用时，必须按照所声明的顺序、个数和数据类型，来传递实际的参数内容。
2）关键字参数
我们在定义函数时，声明了函数的参数后，在调用函数时我们不仅传递实际的参数内容，同时还将参数名字与之对应，这样保证传递进来的参数如果没有按照函数定义时规定的顺序，也能够正确传递进来。
3)默认参数
我们在定义函数时，声明函数的参数，同时可以给这些参数进行赋值，在调用该函数时，如果没有给对应参数传递实参，那么python会默认使用定义函数时所声明的默认参数值。
4）可变参数
普通可变参数：我们在定义函数时，因为参数过多或者不确定具体参数列表的个数，我们还可以采用可变参数来实现。可变参数表示从0到N个参数，采用*参数名
注意，此时可变参数类型，为tuple元组类型。
关键字可变参数：我们在定义函数时，通过使用**参数名，来表示0到N个参数，在外部调用函数时，将这些可变参数组装成一个字典。
例如：

def person(name, password, **kw):
print(‘用户名{0}，密码{1}，其他内容{2}’.format(name,password,kw))

person(‘tom’,’123456’)
用户名tom，密码123456，其他内容{}

person(‘simth’, ‘12345’,sex=’男’, address=’郑州’)
用户名simth，密码12345，其他内容{‘sex’: ‘男’, ‘address’: ‘郑州’}

特殊用法：
在函数定义中，如果单独使用一个号，意味着该后面的参数，为关键字参数声明时必须使用的名字。在调用该函数时，关键字参数的名字要与函数定义中号后面的参数名称一致。例如：
def person(name, password, *, city, sex):
print(‘用户名{0}，密码{1}，其他内容{2}{3}’.format(name, password, city, sex))

person(‘simth’, ‘12345’,sex=’男’, city=’郑州’)

4、局部变量与全局变量
1）Python中，函数在调用的时候Python才会给函数一个内存地址，用来执行函数内部的相关代码。
2)Python程序在运行时，首先会将程序最外部的变量载入内容，并给其分配相关的内存空间。我们成这种变量为全局变量。
3）函数中声明的变量，只有在函数被执行时，才会被分配内存空间进行执行。我们成这种变量为局部变量。
4）Python程序执行的过程中，执行函数时首先会检查其函数内部的局部变量是否存在，如果存在就调用它，如果不存在就从外部全局变量中进行调用，如果仍然没有发现相关内容，则报错。
5）globals()与locals()
在Python中，通过globals()观察全局作用域中内容，包括：内置函数、自定义函数、全局变量；通过locals()观察局部作用域中的内容，包括：局部变量，嵌套函数（内部函数）inner
6）当函数中出现嵌套情况时，最内层函数执行调用变量，会采用“就近原则”即先查找自身范围内的变量，如果没有找到，则向上查找，直到程序最外部结束。

7）global与nonlocal
global用来修饰局部变量时，表示该变量将作为全局变量来使用。即不仅在函数中可以被调用，同时在文件最外部也可以被调用。
nonlocal用来修饰嵌套函数内部的变量，对父级作用域（外层函数）中的变量进行引用和修改，那么从该层(包括父级)以下都将调用该变量的值。注意，该关键字只能影响其直接父级函数。
使用global关键字修饰的变量之前可以并不存在
而使用nonlocal关键字修饰的变量在嵌套作用域中必须已经存在。

总结：在函数中定义的变量，我们成为局部变量。包括在函数内部定义的变量、作为参数传递进来的变量。局部变量起作用的范围，我们称为局部作用域。因为Python在调用局部变量时，需要给它分配内存地址，我们称为这个地址为局部命名空间。
在文件最外层定义的变量，我们成为全局变量，可以在文件中的任意位置进行调用。全局变量起作用的范围，我们称为全局作用域。Python在载入文件时，就需要给全局变量分配内存地址，这个地址我们称为全局命名空间。

Python运行时处理变量的步骤：
1、从Python解释器开始执行之后，就在内存中开辟里一个空间，每当遇到一个变量的时候，就把变量名和值之间对应的关系记录下来，但是当遇到函数定义的时候，解释器只是象征性的将函数名读入内存，表示知道这个函数存在了，至于函数内部的变量和逻辑，解释器根本不关心。
2、函数执行时，函数中的变量会储存在新开辟出来的内存中，函数中的变量只能在函数内部使用，并且会随着函数执行完毕，这块内存中的所有内容也会被清空

6、闭包
1）闭包格式

def outer():
print(‘我是外部函数’)
def inner():
print(‘我是闭包函数’)
return inner

f = outer()
f()
通过对例子的观察，我们发现闭包函数的写法有如下特点：
1、闭包函数作为内部函数，嵌套在外部函数中
2、闭包函数会作为外部函数的返回值，进行返回。返回闭包函数时，只返回函数名，不带括号。
3、外部调用时，首先要执行外部函数，然后将结果赋值给一个变量，此时该变量对应了闭包函数。
4、我们可以通过closure内部函数可以观察到，闭包函数的存在。前提是，闭包函数要与外部函数进行数据传递或调用。
总结：闭包使用的数据，必须是本身定义或外部函数中定义的相关变量，通常需要调用外部函数的数据，形成一个封闭的函数结构。不能与外部数据（全局变量）发生交流。
2）闭包的应用—装饰器
1、概念：装饰器本身仍然是一个函数。是我们对闭包应用的一种具体实现。
2、作用：装饰器可以在不改变函数本身功能的情况下，为函数添加额外的功能。诸如：日志、测试、性能检测等。装饰器同样也是返回一个函数对象。

使用格式：@+装饰器名称

start_time = time.time()
f(var)
time.sleep(0.3)
end_time = time.time()
print(‘此函数执行时间为{0}’.format(str(end_time-start_time)))
return inner

@outer # 此处就是装饰器的声明，这种方法我们成为语法糖

Python对XML文件的解析
1、XML解析的两种方式：DOM与SAX
DOM：文档对象模型，是Python处理XML文件的标准模式。一个DOM的解析器在解析XML文档时 ，一次性读入整个文档，把文档中的所有元素（标签和文本）保存在内存中的一个树形结构，然后我们可以利用DOM提供的各种函数，来读取或修改文档中的内容。
SAX：是一种时间驱动的API，利用SAX解析XML 。主要使用解析器负责读取XML文档，并向事件处理器发生事件，比如：元素开始、元素结束。常用在1）大型文档的处理，2）只需要文件的部分内容，或者从文件中得到特定的部分3）建立自己的对象模型。
dom与sax对比：
1、DOM解析需要读取整个文档到内存中，并且将所有元素保存到树形结构。那也就意味着，可以根据树形结构导航到任意节点，获取其相关内容。
2、SAX解析文档，将文档边读入，边解析的方式，不需要一次性读入全部文档内容，只需要读入一部分即可。同时，因为其边读入边解析的工作方式，能够适应大文件的解析，提高解析效率。

11、包与模块
1）模块
基本概念：在Python中进行Python定义和声明的文件，以.py结尾。使用import或from import导入的文件，我们成为模块。
2）模块的优势
首先，使用模块，可以将常用的功能和变量，统一分类存储在不同的文件中，在使用时进行导入。提供了代码的可维护性。
其次，使用模块因为将常用功能进行了统一调用，提供了代码的复用性
最后，解决了在同一个文件中，方法和变量重名的问题。
3）导入时的问题
如果导入了与文件同名的函数（方法名相同，函数个数相同）时，需要使用模块名调用函数，与文件中的函数进行区分，例如：
import my
my.show()
def show():
pass
show()
4)模块可以在Python文件中，实现多次import导入，但是只有第一次导入时才真正将模块中的内容运行、方法载入，其他再进行导入时，Python解析器会先检测是否已经载入该模块，如果载入就不在执行相关操作。
总结：
模块载入一共分三个步骤：
1、为模块的导入，创建命名空间（划分新的内存地址），被载入的文件按照全局作用域来使用。
2、模块载入后，将在上一步操作中创建的命名空间中，执行模块中的代码
3、创建一个模块的名字，来引用模块所在的命名空间。my.show()

5)当使用from import引入多个模块时，同时多个模块存在同名函数，那么在程序中调用该函数时，以程序导入的最后一个模块的函数为准。

包：
1）概念：包在Python中定义为可以包含多个模块文件的目录，其中必须要包含一个名为“init.py”的文件，用来初始化包。
2）包在调用的使用，首先会初始化init.py文件，主要功能为包中模块运行时提供所需要的相关数据，比如：数据库用户名称、密码、地址等内容。

12、异常
1）概念：指因程序出错在正常的执行流程之外，采取的行为。异常是一个事件，会在程序的执行过程中执行，并影响程序的执行。
2）异常的阶段
第一、产生异常（比如：代码错误、业务逻辑错误）
第二、处理异常（比如：将异常提示更加可理解、增强代码的健壮性）
3）常见的异常类
Exception：Python中所有异常的父类。
NameError：访问了一个未声明的变量。
keyError:访问一个不存在的字典key
IndexError：索引超出序列的范围
ValueError：传给函数的参数类型不正确。
IOError：文件操作时输入输出错误

8）使用异常机制的优势
第一，把错误控制与业务逻辑进行区分，增强程序的可读性和健壮性。
第二，代码结构更加清晰，复杂的业务流程的实现更加便利
